# Hello World
```
$print("Hello, World!")
```
____

```
var = "Hello, World"+"!"
$print(%var%)
```
____

```

fn add(n, n1) {
  return %n%+%n1%;
}
$add(420, 69)
```
____

```
class Program {
  fn main() {
    $input(x)
    $print(x)
  }
}
c = Init(Program.main())
```
